# netcloudmusic

## 说明
- 技术栈：vue3 + vant + better-scroll
- 线上预览：https://neteasecloudmusic.vercel.app

## 主要完成功能
- 播放：列表循环播放，单曲循环播放，随机循环播放，播放上一首，播放下一首
- 歌词滚动高亮
- 进度控制：时间进度条拖动，歌词时间进度拖动
- 声音大小拖动控制
- 音乐下载
